# HR Agency System

Система управления кандидатами для HR-агентства.

## Возможности

### Публичная часть
- Форма подачи резюме с загрузкой файлов (PDF, DOC, DOCX)
- Автоматическое сохранение в базу данных

### Админ-панель
- Управление кандидатами (просмотр, редактирование, удаление)
- Фильтрация по статусам и группам
- История изменения статусов
- Управление тестами и вопросами
- Управление должностями
- Управление группами кандидатов
- Редактирование раздела "О компании"
- Настройки системы

### Личный кабинет кандидата
- Просмотр статуса заявки
- Прохождение тестирования
- Доступ к материалам "О компании" после теста
- Контакты HR

## Установка

### Требования
- PHP 7.4 или выше
- MySQL 5.7 или выше
- Apache/Nginx с mod_rewrite

### Шаги установки

1. **Скопируйте файлы** на ваш сервер в директорию веб-сервера

2. **Создайте базу данных** и импортируйте структуру:
   ```bash
   mysql -u root -p < database.sql
   ```

3. **Настройте подключение к БД** в файле `includes/config.php`:
   ```php
   define('DB_HOST', 'localhost');
   define('DB_NAME', 'hr_agency');
   define('DB_USER', 'ваш_пользователь');
   define('DB_PASS', 'ваш_пароль');
   ```

4. **Настройте URL сайта**:
   ```php
   define('SITE_URL', 'http://ваш-домен.ru');
   ```

5. **Запустите установщик** (создаст администратора):
   - Откройте в браузере: `http://ваш-домен.ru/install.php`
   - Или в командной строке: `php install.php`

6. **Установите права на директории**:
   ```bash
   chmod -R 755 uploads/
   chown -R www-data:www-data uploads/
   ```

7. **Удалите install.php** после успешной установки!

8. **Войдите в админ-панель**:
   - URL: `http://ваш-домен.ru/admin/`
   - Логин: `admin`
   - Пароль: `admin123`

**ВАЖНО:** Сразу после установки смените пароль администратора!

## Статусы кандидатов

- **Новая заявка** - только что подана
- **На рассмотрении** - HR просматривает
- **Приглашён на собеседование** - назначено интервью
- **Тестирование** - проходит тест
- **Одобрен** - успешно прошёл все этапы
- **Отклонён** - не подошёл
- **Принят на работу** - оформлен

## Группы кандидатов

1. **Кандидаты** (системная) - все новые заявки
2. **Постоянные сотрудники** (системная) - оформленные работники
3. Можно создавать свои группы

## Безопасность

- Все входные данные очищаются
- Пароли хешируются (bcrypt)
- Защита от SQL-инъекций (PDO prepared statements)
- CSRF защита
- Ограничение типов загружаемых файлов
- Запрет выполнения PHP в uploads/

## Поддержка

При возникновении проблем проверьте:
1. Права доступа к директориям
2. Параметры подключения к БД
3. Версию PHP (минимум 7.4)
4. Наличие расширений: PDO, pdo_mysql, mbstring
